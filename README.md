Loading Circle Rat
==================

Replaces the loading circles on various websites with a rotating rat.

Currently Supported:
- YouTube
- Canvas Video and Documents (***instructuremedia.com*** & ***canvadocs.instructure.com***)
- Rooster Teeth
- AniWatch (***megacloud.tv***)
- Crunchyroll (***static.crunchyroll.com***)
- Primve Video
